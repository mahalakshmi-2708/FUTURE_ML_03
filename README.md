## 🎯 Objective
To build a machine learning system that automatically screens resumes, ranks candidates based on job role suitability, and identifies missing skills.

---

## 📂 Dataset
A sample dataset of resumes and a job description was used to simulate real-world candidate screening.

---

## ⚙️ Tools & Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- TF-IDF Vectorizer  
- Cosine Similarity  

---

## 🔍 Approach
1. Collected resume text data and job description  
2. Cleaned and preprocessed text (lowercasing, removing special characters)  
3. Converted text into numerical features using TF-IDF  
4. Calculated similarity scores between resumes and job description  
5. Ranked candidates based on similarity scores  
6. Identified missing skills for each candidate  

---

## 📈 Output
- Ranked resumes based on job relevance  
- Similarity scores for each candidate  
- Missing skills for each resume
